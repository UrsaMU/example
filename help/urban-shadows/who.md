# Who — Urban Shadows

The `+who` command shows who is actively playing and gives a quick status
snapshot for each character.

## Commands

### +who

List all approved characters with playbook, harm, and corruption.

    +who

### +who/view \<name\>

View one character's public details: playbook, harm, corruption, and circle
status. Accepts a name prefix (case-insensitive).

    +who/view Ghost
    +who/view bl

---

## Quick Reference

| What you want to do              | Command            |
| -------------------------------- | ------------------ |
| See all active characters        | `+who`             |
| View one character's public info | `+who/view <name>` |
