# Urban Shadows

A game of dark urban fantasy powered by the Apocalypse. You play citizens of a
city teeming with supernatural factions — vampires, fae, mages, and worse — all
negotiating power, favors, and survival.

## Topics

- **help chargen** — Creating your character
- **help moves** — Triggering moves and rolling dice
- **help tracking** — Harm, armor, and corruption
- **help advancement** — XP and advances
- **help circles** — Circle status and faction affiliations
- **help debts** — Tracking favors and obligations
- **help session** — End-of-session XP questions
- **help who** — Active character roster
- **help scene** — Room scene descriptions
- **help downtime** — Between-session downtime actions
- **help orgs** — City factions and organizations
- **help jobs** — Job requests and staff review
- **help npcs** — [Staff] NPC harm tracker
- **help fronts** — [Staff] Fronts & doom clocks
- **help gm** — [Staff] AI Game Master

## Quick Reference

| What you want to do        | Command                  |
|----------------------------|--------------------------|
| See your character sheet   | `+chargen`               |

| Roll a stat                | `+roll <stat>`           |
| Roll with an explicit value| `+roll/free <stat>=<n>`  |
| Mark harm                  | `+harm/mark`             |
| Mark XP                    | `+xp/mark`               |
| View debts                 | `+debts`                 |
| View circles               | `+circles`               |
| See active characters      | `+who`                   |
| Read current scene         | `+scene`                 |

| List organizations         | `+org/list`              |
| View your jobs             | `+jobs`                  |
| List NPCs [staff]          | `+npc/list`              |
